# API Resiliency Testing with Specmatic

## API Resiliency Overview Video
[![Watch the video](https://img.youtube.com/vi/ha0DDRArM7E/hqdefault.jpg)](https://www.youtube.com/watch?v=ha0DDRArM7E)

Start the full stack using Docker Compose:

```shell
docker compose up
```

Clean up:
```shell
docker compose down -v
```

## Time required to complete this lab:
10-15 minutes.

## Prerequisites
- Docker is installed and running.
- You are in `labs/api-resiliency-testing`.
